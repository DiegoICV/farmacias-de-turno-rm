Farmacias en turno


El estado inicial del mapa muestra todas las Farmacias de la Region Metropolitana, al utilizar el filtro se muestran solo las farmacias de turno ( En horario de atencion)

Observación: la rest utilizada NO PROVEE las farmacias de turno. Solo las farmacias Abiertas a la hora actual

La api que si muestra las farmacias de turno se encuentra en: https://farmanet.minsal.cl/maps/index.php/ws/getLocalesTurnos
